Netpluspay Woocommerce Payment Plugin allows you to accept payment on your Woocommerce store via Visa Cards, Mastercards and Internet banking platform.




== Description ==

> This is a Netpluspay Payment Plugin for Woocommerce.
> Allows your website ready to collect payments easily via credit/debit cards/internet banking on your ecommerce store in safe way. 


With Netpluspay Woocommerce Payment Plugin, you will be able to accept the following payment methods in your shop:

* __MasterCards__
* __Visa Card__
* __Internet Banking__






== Installation ==

= Automatic Installation =
* 	Login to your WordPress Admin area
* 	Go to "Plugins > Add New" from the left hand menu
* 	In the search box type "Netpluspay Woocommerce Payment Gateway"
*	From the search result you will see "Netpluspay Woocommerce Payment Gateway" click on "Install Now" to install the plugin
*	A popup window will ask you to confirm your wish to install the Plugin.

= Note: =
If this is the first time you've installed a WordPress Plugin, you may need to enter the FTP login credential information. If you've installed a Plugin before, it will still have the login information. This information is available through your web server host.

* Click "Proceed" to continue the installation. The resulting installation screen will list the installation as successful or note any problems during the install.
* If successful, click "Activate Plugin" to activate it, or "Return to Plugin Installer" for further actions.

= Manual Installation =
1. 	Download the plugin zip file
2. 	Login to your WordPress Admin. Click on "Plugins > Add New" from the left hand menu.
3.  Click on the "Upload" option, then click "Choose File" to select the zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
4.  Activate the plugin.
5. 	Open the settings page for WooCommerce and click the "Payment Gateways," tab.
6. 	Click on the sub tab for "Netpluspay Payment Gateway".
7.	Configure your "Netpluspay Payment Gateway" settings. See below for details.



= Configure the plugin =
To configure the plugin, go to __WooCommerce > Settings__ from the left hand menu, then click "Payment Gateways" from the top tab. You should see __"Netpluspay Payment Gateway"__ as an option at the top of the screen. Click on it to configure the payment gateway.

__*You can select the radio button next to the Netpluspay Payment Gateway from the list of payment gateways available to make it the default gateway.*__

* __Enable/Disable__ - check the box to enable Netpluspay Payment Gateway.
* __Production__ - check the box  when you have been told to switch to live mode
* __Description__ - controls the message that appears under the payment fields on the checkout page. Here you can list the types of cards you accept.
* __Test Merchant ID__  - enter your test Merchant ID, this is gotten from your account page on merchants.netpluspay.com.
* __Merchant ID__  - enter your Merchant ID, this is gotten from your account page on merchants.netpluspay.com.
* Click on __Save Changes__ for the changes you made to be effected.
